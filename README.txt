EMPÓRIO PET RH - VERSÃO 4

ARQUIVOS:
- index.html
- style.css
- app.js
- supabase_setup.sql

O QUE ESSA VERSÃO ENTREGA:
- dashboard inicial
- lista visual de funcionários
- busca por nome, CPF, cargo, telefone
- filtros por status, setor e cargo
- cadastro completo
- edição de funcionários
- foto do funcionário
- status: ativo, experiência, afastado, desligado
- data de admissão
- salário
- tipo de contrato
- contato de emergência
- observações
- alertas de experiência (90 dias)
- aniversariantes do mês
- configuração do Supabase direto no sistema

COMO SUBIR NO GITHUB PAGES:
1. Apague os arquivos antigos do repositório ou substitua pelos novos.
2. Envie index.html, style.css e app.js para a raiz do repositório.
3. Faça commit.
4. Aguarde a publicação do Github Pages.

COMO PREPARAR O SUPABASE:
1. Abra o SQL Editor do Supabase.
2. Execute o arquivo supabase_setup.sql.
3. No sistema, clique em Configurações.
4. Cole a URL e a ANON KEY do projeto.
5. Salve.

IMPORTANTE:
- O bucket padrão de fotos é employee-photos.
- Essa versão usa políticas abertas para facilitar a publicação no Github Pages.
- Se quiser, depois posso te entregar a V5 com:
  - documentos do funcionário
  - histórico de advertências/atestados/férias
  - tela individual do colaborador
  - exportação PDF
  - login por perfil
